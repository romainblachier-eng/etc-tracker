# Guide d'Installation - Mounjaro Magazine

## Prérequis

- WordPress 5.0 ou ultérieur
- PHP 7.2 ou ultérieur
- Accès au panneau d'administration WordPress
- Une connexion Internet pour les mises à jour

## Étapes d'Installation

### 1. Télécharger le Thème

#### Option A: Via le dossier themes
1. Téléchargez le dossier `mounjaro-magazine`
2. Uploadez-le dans `/wp-content/themes/` via FTP ou SFTP
3. Assurez-vous que le dossier est correctement transféré

#### Option B: Via ZIP
1. Compressez le dossier `mounjaro-magazine` en ZIP
2. Dans l'admin WordPress: Apparence > Thèmes > Ajouter
3. Cliquez sur "Télécharger un thème"
4. Sélectionnez le fichier ZIP
5. Cliquez sur "Installer maintenant"

### 2. Activer le Thème

1. Allez à **Apparence > Thèmes**
2. Trouvez "Mounjaro Magazine"
3. Cliquez sur "Activer"

### 3. Configuration Initiale

#### Site Title & Tagline
1. Allez à **Paramètres > Général**
2. Entrez votre titre et slogan
3. Cliquez sur "Enregistrer les modifications"

#### Personnalisation
1. Allez à **Apparence > Personnaliser**
2. Configurez:
   - **Identité du site**: Logo et titre
   - **Couleurs du thème**: Couleurs primaires et secondaires
   - **Menus**: Créez et assignez vos menus

#### Menus
1. Allez à **Apparence > Menus**
2. Cliquez sur "Créer un nouveau menu"
3. Nommez votre menu (ex: "Menu Principal")
4. Ajoutez des pages, catégories ou liens personnalisés
5. Cochez "Afficher dans le Menu Principal"
6. Cliquez sur "Créer le menu"

#### Widgets
1. Allez à **Apparence > Widgets**
2. Configurez:
   - **Sidebar**: Widget latéral
   - **Footer 1-3**: Zones footer
3. Glissez-déposez les widgets selon vos besoins

### 4. Contenu Initial

#### Créer des Articles
1. Allez à **Articles > Ajouter**
2. Remplissez le titre et le contenu
3. Assignez une catégorie et des tags
4. Uploadez une image en vedette
5. Publiez

#### Créer des Catégories
1. Allez à **Articles > Catégories**
2. Cliquez sur "Ajouter une nouvelle catégorie"
3. Remplissez le formulaire
4. Cliquez sur "Ajouter une nouvelle catégorie"

#### Créer des Pages Statiques
1. Allez à **Pages > Ajouter**
2. Remplissez le titre et le contenu
3. Publiez

### 5. Configuration Avancée

#### Paramètres de Lecture
1. Allez à **Paramètres > Lecture**
2. Configurez:
   - **Nombre maximum de messages**: Articles par page (recommandé: 10)
   - **Fil de syndication**: Articles dans le flux (recommandé: 10)

#### Commentaires
1. Allez à **Paramètres > Commentaires**
2. Activez les commentaires si souhaité
3. Configurez la modération

#### Permaliens
1. Allez à **Paramètres > Permaliens**
2. Choisissez "Nom de l'article" pour de meilleurs URLs
3. Cliquez sur "Enregistrer les modifications"

## Dépannage

### Le thème ne s'affiche pas correctement
- Vérifiez votre version de WordPress (5.0+)
- Vérifiez votre version PHP (7.2+)
- Désactivez les plugins conflictuels temporairement
- Videz le cache du navigateur

### Les images ne s'affichent pas
- Vérifiez les permissions du dossier `/wp-content/uploads/`
- Assurez-vous que les fichiers sont correctement uploadés
- Vérifiez l'URL du site dans Paramètres > Général

### Les menus ne s'affichent pas
- Allez à **Apparence > Menus**
- Assurez-vous qu'un menu est assigné au "Menu Principal"

### CSS ou JavaScript ne charge pas
- Videz le cache (navigateur et plugins)
- Vérifiez les permissions des fichiers
- Vérifiez la console du navigateur pour les erreurs

### Les commentaires ne s'affichent pas
- Allez à **Paramètres > Commentaires**
- Cochez "Autoriser les commentaires"
- Vérifiez la modération des commentaires
